# 《燁程的異世界初醒》腳本大綱（15s｜12 幕）

## 場景概述
- 燁程在陌生異世界的苔蘚岩地甦醒，仍穿著辦公室衣著，與潮濕荒蕪的森林形成強烈對比。
- 天空被紫藍裂縫撕開，巨大的光之電路在雲層上流動，散發不安的電磁脈衝。
- 林中充滿帶刺藤蔓、濕潤霧氣與未知獸吼，暗處逐步亮起一雙雙綠光眼睛。
- 骨甲外露的瘦長怪物從樹影現身，團團包圍燁程，逼迫他啟動光迴路術師的職業初始化。

## 逐幕分鏡
1. **0.0–1.2s**：中近景斜俯視，燁程側躺在覆滿苔蘚與砂礫的岩地，手指抓緊地面，表情困惑甦醒。
2. **1.2–2.4s**：側面中景，他撐著地面慢慢坐起，整理領帶，看見周遭陌生森林露出迷惘。
3. **2.4–3.6s**：仰角望向天空，紫藍裂縫與光之電路板在雲層上閃動，電弧貫穿整片天幕。
4. **3.6–4.8s**：中近景，燁程皺眉吐槽，喃喃抱怨「這 shader 也太浮誇」，語氣半驚半慌。
5. **4.8–6.0s**：POV 環視，潮濕岩壁、黑色高樹、帶刺藤蔓與薄霧交錯，畫面邊緣帶輕微暈影。
6. **6.0–7.2s**：遠景，森林深處傳來低沉咆哮，空氣震動，枝葉跟著抖動落下水珠。
7. **7.2–8.4s**：樹影特寫，一雙狹長綠光眼睛逐步亮起，靜靜盯著燁程。
8. **8.4–9.6s**：鏡頭後拉，更多綠光眼睛在不同樹影間依序點亮，像錯誤訊號同時跳紅。
9. **9.6–10.8s**：怪物踏出樹線，骨甲外露、下顎過長的瘦長怪物踩碎泥土，露出銳利指爪。
10. **10.8–12.0s**：側視中景，三、五、更多怪物從不同方向靠近，慢慢半圓形包抄燁程。
11. **12.0–13.2s**：骨甲與血鏽特寫，外翻骨板、乾裂血跡與掉落的紅褐碎屑，呼吸聲粗重黏膩。
12. **13.2–15.0s**：回到燁程近景，他滿頭冷汗後退，視野上方浮出半透明 UI：「【職業初始化中：光迴路術師（Lv.1）】」。

## 聲音設計指示
- 前段以潮濕森林環境音、遠方風聲與燁程衣料摩擦聲建立陌生感。
- 天空電路板伴隨低頻電流嗡鳴與偶發閃爍電弧，與燁程吐槽時留出空間。
- 怪物眼睛點亮時加入細微錯誤提示音與節奏性低頻脈衝，逐步堆疊張力。
- 群怪現身後提升骨甲摩擦、泥地破裂與濕潤呼吸聲，最後 UI 彈出時加上透明介面啟動音與數據流動聲。

## 後製備註
- 燁程服裝保持現代上班族襯衫與領帶，但附著苔蘚與泥水細節，強化穿越落差。
- 天空裂縫需持續流動，光之電路板軌跡與地表苔蘚上亮起的光點同步閃爍。
- 怪物的骨甲與血鏽需要分層貼圖，呼吸時骨板微幅張闔並掉落粉末粒子。
- UI 提示採透明玻璃質感，字體以虛擬光線雕刻呈現，避免遮擋燁程表情。
