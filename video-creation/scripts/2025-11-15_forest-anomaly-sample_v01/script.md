# 異界編譯 EP7｜森林異常與獵骨獸樣本 — 原始腳本要點

- 15 秒｜12 幕分鏡（巡林出任務 → 遭遇 → 樣本電路紋 → Error Log 伏筆）
- 幕 1（0–1s）：濕冷森林巡邏列隊。俯角跟拍隊伍踏過泥水，燁程排在中段。
- 幕 2（1–2s）：俯視特寫看異常密集爪痕與血跡腳印，隊長低聲評估密度升高。
- 幕 3（2–3s）：仰視紫藍電路天空，特定區塊閃爍失衡，環境聲被「降噪」。
- 幕 4（3–4s）：樹叢傳來骨甲撞擊與低吼，螢綠瞳孔亮起，隊伍準備戰鬥。
- 幕 5（4–5s）：標準陣式光彈射向獵骨獸，燁程手環亮起 Curve Bolt 預覽。
- 幕 6（5–6s）：協同擊殺，燁程彎射鑽入骨縫，獵骨獸倒地濺泥，隊長吩咐留樣本。
- 幕 7（6–7s）：隊員圍檢樣本，清理骨板，燁程蹲下喘息。
- 幕 8（7–8s）：極近特寫骨板電路紋重複 template。
- 幕 9（8–9s）：切燁程主觀視角，UI 偵測模式高亮 pattern。
- 幕 10（9–10s）：Error Log 半透明視窗閃現亂碼後抖動。
- 幕 11（10–11s）：外部視角，只有燁程看到異常，隊友疑惑，他勉強回應。
- 幕 12（11–15s）：拉遠見森林各處複製 pattern，遠處綠眼再現，隊長下令撤退，燁程耳邊仍有 UI 雜訊。
