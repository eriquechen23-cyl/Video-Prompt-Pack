# 2025-02-17 Lightning Chain Barrage v01 Script

Master（15s｜雷導術式×藍白離子電弧｜16:9｜24fps｜濕潤石板巷夜雨後）
「濕潤石板街映出霓虹殘影，你站於圓形法陣中央，右手指尖帶著藍白電暈畫圈蓄能，空氣充滿金屬與臭氧味，胸口隨低頻嗡鳴起伏。鏡頭以 Steadicam 低角度貼近手勢，隨電弧牽引繞場；服裝為導電纖維戰袍，袖口嵌入銀色導環，腰際掛藍光電容器。霧氣輕貼鏡面，背景遠處偶有閃光倒影。無字幕、無商標/Logo/水印。」

Act 1（0.0–4.8s）
意圖：點燃圓弧與首段雷鏈節奏
基調：聚焦、蓄壓
節奏域：1.0–1.2s
美術要點：指尖電弧、地紋亮起、藍白主調

Beat 1（0.0–1.2s）
重點：近手特寫點火
Blocking：右手指尖沿圓軌劃過空氣，左手托腕，口中吐霧，肩帶微顫
Camera：
- 運鏡：微俯 20° 推近 30cm
- 焦段/機位：60mm 特寫
- 對焦：指尖→電弧尾焰
光影：法陣光由暗至亮 30%，手上反射補光 5200K
聲音：低頻嗡鳴漸起，微弱 crackle 前景
轉場：前一場以光閃剪進入指尖
VFX：指尖貝塞爾曲線電弧生成，細小火花噴濺，空氣折射震動

Beat 2（1.2–2.4s）
重點：俯拍地紋亮起
Blocking：雙腳踩穩，法陣紋理擴散，細火花竄動，腳邊水珠跳起
Camera：
- 運鏡：垂直升 1m 俯視
- 焦段/機位：35mm overhead
- 對焦：法陣外圈→中心掌心
光影：地紋藍白環形掃過地面，濕地反射
聲音：細密電刺，嗡鳴加層 120 BPM 低拍
轉場：遮擋剪（手臂抬起掠鏡）
VFX：地面 Signed Distance Field 輸出圓陣光，微火花粒子跳動

Beat 3（2.4–3.6s）
重點：半身側拍，電弧飛向第一目標
Blocking：右腕內旋，食中指並攏推出，視線鎖定前方第一節點
Camera：
- 運鏡：側向 45° 跟隨 0.5m
- 焦段/機位：40mm 半身
- 對焦：掌心→外放電弧
光影：電弧亮度峰值 0.9，背景保持 0.4
聲音：雷鏈首次爆發強拍，sub_thump 貼合
轉場：形狀配對剪（圓弧接到遠方節點）
VFX：貝茲曲線電弧拉向前方目標，殘影延遲 2 frame

Beat 4（3.6–4.8s）
重點：長焦拉近電弧跳至第二目標
Blocking：身體向右微轉，左手指引下一節點
Camera：
- 運鏡：長焦 85mm 拉近 2m
- 對焦：第一節點殘影→第二節點爆
光影：電弧殘影拖曳，背景冷暗
聲音：連鎖劈啪與低頻連擊
轉場：光閃剪，第一節點過曝遮罩
VFX：電弧「跳」至第二目標並留 3 frame 殘影尾巴

- 運鏡：空拍下壓 2m

Beat 6（6.0–7.2s）
重點：主觀視角 HUD 連結
Blocking：視角內 HUD 節點亮起，手在畫圈，畫面輕晃

- 運鏡：主觀視角微抖
- 焦段/機位：虛擬 35mm POV
- 對焦：HUD 節點→遠方鏈路

VFX：HUD overlay 與雷鏈節點對應，畫面加入輕微 chroma noise

Beat 7（7.2–8.4s）
重點：環繞 120° 多鏈同爆
Blocking：角色核心不動，雷鏈四向炸開，光暈脈衝
Camera：
- 運鏡：環繞 120° 右→左，速度加速
- 焦段/機位：32mm 肩平
- 對焦：角色→外圈爆點
光影：光暈過曝 1.1，漸層回落 0.5
聲音：多點爆裂與合成鼓強拍
轉場：形狀配對剪（圓形光暈接下一廣角）
VFX：多鏈同時爆的過曝 bloom，殘影緩退

Beat 8（8.4–9.6s）
重點：廣角掃街顯示遠處連鎖

- 運鏡：廣角 18mm 水平掃 60°
- 對焦：街道鏈路→反射

聲音：遠距回聲與城市環境混合
轉場：遮擋剪（路燈柱掠過）
