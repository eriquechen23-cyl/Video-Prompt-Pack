# 2025-02-19 Parkour Outfit Switch v02 Script

Master（15s｜城市夜景跑酷 × 特效換裝｜16:9｜30fps｜混合實拍＋VFX）

「夜間濕潤巷弄的跑酷者只靠 VFX 變裝。8 幕連貫、每幕 1.0–1.3s 的密集節奏以彩煙、墨溶、像素爆、光絲纏、卡牌擦、水膜刷新、羽化逆光、殘影回帖推進。基底服裝為貼身中性色排汗衣褲，胸前貼 3 個追蹤點；所有換裝沿用同版型不同材質／色彩。主鏡頭為貼身手持，輔以兩個固定鏡位提供保險角度。節奏以 108 BPM 設定，每次換裝落在強拍上，聲音層包含鞋底摩擦、布料刷動、風切、雨滴與終點低頻『咚』。」

## Scene Breakdown

### Scene 1｜起跑煙霧 Poof（00.0–01.3）
- **動作**：跑者於濕地面加速起跑衝向鏡頭，左袖甩過鏡頭造成擦拭遮擋。
- **換裝**：黑色貼身排汗 T → 白色貼身 T，褲裝不變。
- **特效**：袖口帶動的彩煙粒子在鏡頭前「嘶」地炸開並迅速成球體遮擋；煙霧內以 Cloth Morph 一拍成形白 T。
- **攝影**：Steadicam 低角 24mm 貼身跟拍，30fps、快門 180°。0.2s 前導加速、0.4s 衝向鏡頭、0.7s 被煙霧遮擋後揭開白 T。
- **聲音／感官**：鞋底摩擦、地面積水噴濺，風貼臉頰的呼聲。
- **追蹤／合成**：胸前三追蹤點供 Garment 投影；煙霧為可重用彩煙預設，加入方向性運動模糊。

### Scene 2｜手背一抹｜墨溶換帽（01.3–02.5）
- **動作**：跑者抬右臂橫抹胸前後繼續前進。
- **換裝**：白 T → 螢光連帽外套（帽沿上翻）。
- **特效**：沿手背軌跡噴出墨滴／油漬貼花，使用 SDF dissolve 從胸前向上長出帽沿與帽身。
- **攝影**：手持 28mm 腰高跟拍，快門 200°，在抹胸瞬間略微 Dolly-in。
- **聲音／感官**：指尖劃過布面發出乾澀摩擦，鼻端有油墨味的呼吸描寫。
- **合成**：墨滴元素使用預設墨溶貼花，對應手勢以 Mesh Warp 校正布面。

### Scene 3｜Kong Vault｜像素爆換（02.5–03.7）
- **動作**：跑者進行 Kong vault 跨越檯面，半空定格 6–8 帧。
- **換裝**：螢光連帽 → 迷彩機能外套。
- **特效**：Freeze-frame 期間身體分解為像素方塊炸裂，再回收組裝成迷彩衣。
- **攝影**：吊臂前導 35mm，由欄杆前景 Whip pan 接至空中定格，再跟至落地。
- **聲音／感官**：呼吸急促、衣料刷風聲；像素破裂配合短促電子啪聲。
- **合成**：像素爆為預先設計粒子系統，可套換材質；使用 Motion Vector 保持身體動勢銜接。

### Scene 4｜牆跑旋身｜光絲纏衣（03.7–04.9）
- **動作**：跑者在牆面踩步後 270° 旋身落地。
- **換裝**：迷彩機能 → 輕薄風衣（淡銀色）。
- **特效**：兩條光絲沿軀幹螺旋纏繞，尾焰掃過即換色，殘留淡光粒尾。
- **攝影**：手持 32mm 與角色一同旋轉，快門 220° 製造方向性模糊，落地時微震。
- **聲音／感官**：項鍊碰撞「叮」、風衣布料飄動。
- **合成**：光絲使用可重用光帶預設，附帶 Glow 與位移貼圖，Garment Warp 校形。

### Scene 5｜門框穿切｜卡牌擦換（04.9–06.2）
- **動作**：跑者 Tic-tac 踢牆轉身穿越門框。
- **換裝**：輕薄風衣 → 無袖背心。
- **特效**：三層分割卡牌由左至右擦過畫面，逐層揭出新服裝。中層帶有金屬反光紋理。
- **攝影**：手持 30mm 跟進門內，門框形成黑邊遮擋，快門 190°。
- **聲音／感官**：門框金屬「咔」、布料掠過手臂的沙沙聲。
- **合成**：卡牌元素為重用卡牌擦換預設，可調整方向；門框邊緣加 Motion Blur，提升隱剪效果。

### Scene 6｜雨點貼鏡｜水膜刷新（06.2–07.5）
- **動作**：跑者貼近鏡頭跨越障礙，左手撥開雨簾。
- **換裝**：無袖背心 → 透明雨衣（可改為銀色風衣視品牌需求）。
- **特效**：鏡面雨滴放大成水膜覆蓋鏡頭；使用 Flowmap 漂洗讓新服裝於水膜內成形。
- **攝影**：手持 35mm 貼近，先拍手臂特寫再後退，快門 210° 捕捉雨滴拖尾。
- **聲音／感官**：雨滴「噠噠」敲鏡、肩頭瞬間清涼感描述。
- **合成**：水膜預設配合 Luma key 及 UV Distort，雨滴粒子與鏡頭水跡合成。

### Scene 7｜下坡滑行｜羽化逆光（07.5–08.9）
- **動作**：跑者腳跟點地在斜坡上滑行。
- **換裝**：透明雨衣／銀色風衣 → 黑金運動套裝。
- **特效**：強逆光下角色成剪影，衣料內部羽化溶解，邊緣 Fresnel 閃光後收束。
- **攝影**：滑軌 40mm 胸高逆光跟拍，快門 170°；鏡頭微向下追。
- **聲音／感官**：衣料刷刷掠風、地面沙粒細響。
- **合成**：羽化粒子沿角色輪廓噴出，內部使用 Luma matte 轉換為黑金材質。

### Scene 8｜終點定格｜殘影回帖（08.9–12.0）
- **動作**：跑者急停、回望鏡頭、微笑點頭。
- **換裝**：殘影回帖成品牌最終版黑金套裝，細節如 Logo 線條與反光。
- **特效**：Echo trails 依鼓點節拍吸回身上，完成後微 Bloom 鎖定。Freeze-frame 1.9s 保留品牌字樣與口號。
- **攝影**：滑軌回拉至正面中景，50mm，快門 180°，Freeze-frame 轉為靜態。
- **聲音／感官**：呼吸吐霧、終點低頻「咚」收尾，遠處城市環境聲漸淡。
- **合成**：殘影預設以時間位移層組成；Freeze 時間繼續輸出輕微光脈動，可在 12–15s Out 段疊品牌資訊。

## Production Notes
- **服裝基底**：貼身中性色衣褲搭配 3 點追蹤標記，確保所有換裝採平面投影 + Mesh Warp 校形。
- **鏡頭配置**：主機手持貼身跟拍；固定機位 A（長焦俯視）與 B（低角廣角）供保險切換。
- **追蹤與動態模糊**：轉身、Whip pan、Freeze 等段落加入方向性模糊以藏剪；所有換裝帧確認手勢遮擋充分。
- **素材庫**：彩煙、墨溶、像素爆、光帶、卡牌、流體水膜、羽化粒子、殘影 Echo 全以可重用預設建立，保留顏色與強度可調參數。
- **聲音設計**：鞋音、布擦、微風、雨滴、電子啪聲按 BPM 108 對拍；終章低頻衝擊與品牌口號同時上揚。
- **安全**：濕地面與斜坡均須防滑處理，Kong vault 與 Gap 區域設軟墊（後期移除）；雨幕與煙霧設備避免影響跑者視線。
